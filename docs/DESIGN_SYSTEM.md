# Py2Learn Design System

Py2Learn uses reusable UI components to keep the app consistent, scalable, and easy to maintain.

## Principles

- Dark, modern, friendly interface
- Mobile-first layout
- Rounded cards and clear spacing
- Cyan as the primary action colour
- Simple readable components
- Reusable before custom

## Components

### Button

Use for links and actions.

Variants:
- `primary`
- `secondary`
- `ghost`
- `danger`

### Card

Use for grouped content, summaries, lesson blocks, and dashboard panels.

### Badge

Use for status, difficulty, XP labels, and categories.

Variants:
- `cyan`
- `purple`
- `green`
- `amber`
- `slate`

### ProgressBar

Use for lesson progress, profile progress, practice progress, and project completion.

### SectionHeader

Use at the top of major sections.

### StatCard

Use for dashboard/profile stats such as XP, streaks, completed lessons, and projects.

## Rule

Before creating new custom styling, check whether an existing UI component can be reused.
